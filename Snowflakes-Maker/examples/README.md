## üìö Available Examples

### 1. `basic-snowflake.gh`
**Difficulty**: Beginner  
**Description**: Simple single snowflake generation with default parameters.

**What you'll learn**:
- Basic component setup
- Understanding outputs
- Previewing geometry
- Saving DNA codes

**Components used**:
- Snowflakes Maker
- Panel (for DNA code)

---

### 2. `multiple-variations.gh`
**Difficulty**: Beginner  
**Description**: Generate 10 variations at once using seed series.

**What you'll learn**:
- Using Series component
- Batch generation
- Comparing variations
- Baking results

**Components used**:
- Snowflakes Maker
- Series
- Bake component

---

### 3. `thickness-exploration.gh`
**Difficulty**: Intermediate  
**Description**: Create a grid of snowflakes with varying thickness.

**What you'll learn**:
- Range parameter control
- Grid layout organization
- Visual comparison
- Exporting for presentation

**Components used**:
- Snowflakes Maker
- Range
- Move
- Grid arrangement

---

### 4. `laser-cutting-export.gh`
**Difficulty**: Intermediate  
**Description**: Prepare snowflakes for laser cutting with proper scaling and layout.

**What you'll learn**:
- Scaling to real-world dimensions
- Nesting for material efficiency
- DXF export preparation
- Layer organization

**Components used**:
- Snowflakes Maker
- Scale
- Orient
- Boundary Box

---

### 5. `3d-printing-ornament.gh`
**Difficulty**: Intermediate  
**Description**: Create a 3D printable snowflake ornament with mounting hole.

**What you'll learn**:
- Extruding surfaces
- Adding mounting features
- STL export setup
- Thickness considerations

**Components used**:
- Snowflakes Maker
- Extrude
- Circle
- Boolean Difference

---

### 6. `collection-generator.gh`
**Difficulty**: Advanced  
**Description**: Generate a themed collection with controlled parameters.

**What you'll learn**:
- Systematic variation
- DNA code management
- Batch export
- Quality control

**Components used**:
- Snowflakes Maker
- Data management
- File export
- List processing
